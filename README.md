SmartSchedule — трекер повторяющихся задач

🔹 SmartSchedule — удобный трекер для автоматизации планирования повторяющихся задач (еженедельных, ежемесячных и т. д.).

🔹 Для кого это?\
Программа идеально подходит для сотрудников, у которых есть регулярные задачи в разные дни. Больше не нужно вручную
собирать расписание на месяц — SmartSchedule сделает это за вас!

🔹 Как это работает?\
• Пользователь указывает длительность рабочего дня.\
• Система автоматически распределяет задачи, учитывая их сроки выполнения.\
• Если задач на день слишком много, их можно гибко переносить (вперёд или назад, в зависимости от настроек
пользователя).\
• Ни одна важная задача не будет упущена — больше не нужно держать всё в голове!

🔹Технологий стэк:\
• Язык: Java 21, Java Core\
• Фреймворки: Spring Boot, Spring Data JPA\
• Базы данных: PostgreSQL, H2 (для тестов)\
• Логирование: SLF4J\
• Удобство разработки: Lombok\
• Тестирование: JUnit, Mockito, Spring Boot Test\
• Инструменты: Docker, Maven, Postman, DBeaver, IntelliJ IDEA

🔹Цель проекта:\
Создать удобный и умный планировщик, который экономит время и снижает нагрузку на пользователя, автоматизируя рутинное
расписание.
